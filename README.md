# CoffeeTalk Mobile App Framework

This project contains the basic framework for a mobile application targeting Android, using Angular for the frontend and Spring Boot for the backend.

## Structure
- `frontend/` - Angular app (displays heading "CoffeeTalk")
- `backend/` - Spring Boot app (basic setup)

## How to Run

### Frontend (Angular)
1. Navigate to `frontend` directory
2. Run `npm install` to install dependencies
3. Run `npm start` to launch the app

### Backend (Spring Boot)
1. Navigate to `backend` directory
2. Run `mvn spring-boot:run` to start the backend server

---

This is a starter template. The frontend will show a heading "CoffeeTalk" when opened. No additional functionality is included yet.
